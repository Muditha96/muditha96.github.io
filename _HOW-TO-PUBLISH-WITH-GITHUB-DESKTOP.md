# Publish with GitHub Desktop (easiest — no file limit)

This folder is CLEAN and ready to upload. Use GitHub Desktop to push it all at once.

## Steps
1. Download & install GitHub Desktop: https://desktop.github.com
2. Open it → sign in to GitHub.
3. File → Clone repository → select **muditha96/muditha96.github.io** → choose a folder (e.g. Desktop) → Clone.
4. Open the cloned folder in File Explorer.
5. If it already has files from your earlier web uploads, you can leave them — copying over will update them.
6. Copy **everything inside this SITE-READY-TO-UPLOAD folder** into the cloned repo folder (choose "Replace files in destination" if asked).
7. Go back to GitHub Desktop — it lists all changes.
8. Bottom-left: type a summary "Publish full portfolio site" → click **Commit to main**.
9. Top: click **Push origin**.
10. Wait ~1 minute, open https://muditha96.github.io — your site is live.

## Notes
- This handles ALL files and folders at once — no 100-file limit, correct paths.
- Future updates: edit files here, copy into the cloned folder, Commit + Push again.
