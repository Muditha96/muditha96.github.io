Project Overview

Public-safe summary of reliability support for an LED bulb assembly machine, covering fault isolation, mechanical checks, sensor checks and recovery actions.

Problem / Requirement

Repeated stoppages and unstable machine behavior affected assembly reliability.

My Role

Supported fault finding, component checks, operator feedback review and recovery actions.

Engineering Approach

Separated symptoms into mechanical, pneumatic, sensor and control categories before validating likely causes through safe checks and repeat testing.
